# ds-section1-project1
👩🏻‍💻codestates section1 project session  
🕹 캐글의 [비디오 게임 세일즈 데이터](https://www.kaggle.com/gregorut/videogamesales)를 활용하여  
🎙다음 분기에 어떤 게임을 설계해야 잘 팔릴까 제안해 보기

### data description
`Name` : 게임의 이름  
`Platform` : 게임이 지원되는 플랫폼  
`Year` : 게임이 출시된 연도  
`Genre` : 게임의 장르  
`Publisher` : 게임 배급사  
`NA_Sales` : 북미 매출액  
`EU_Sales` : 유럽 매출액  
`JP_Sales` : 일본 매출액  
`Other_Sales` : 기타지역 매출액
